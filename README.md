## Como executar localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/rianbene/Squad-2-UmSushi.git
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Rode o projeto:
   ```bash
   npm run dev
   ```

> Certifique-se de que você tenha o Node.js instalado.
